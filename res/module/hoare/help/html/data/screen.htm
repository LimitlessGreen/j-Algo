Der Arbeitsbereich

Der Arbeitsbereich ist untergliedert in sechs Bereiche, die Ihnen Zugriff auf alle wesentlichen Funktionen anbieten, sowie aktuelle und relevante Informationen darstellen:

  1. Toolbar

    Hier können die wichtigsten Einstellungen vorgenommen werden.
    Mehr Informationen hierzu: Anzeigeoptionen  

  2. Beweisbaum

    Hier wird der Beweisbaum dargestellt.
    Mehr Informationen hierzu: Der Beweisbaum

  3. Quellcode

    Hier wird der Quellcode des C0-Programmes angezeigt. Dieser kann jederzeit verändert und neu geparst werden.
    Mehr Informationen hierzu: Der Quellcode

  4. Event-Log

    Hier werden Sie über aktuelle Ereignisse bzw Fehler beim Aufbau des Beweisbaumes informiert.
    Mehr Informationen hierzu: Der Event-Log

  5. Verifikationsregeln

    Hier wird die Liste der für das Hoare-Kalkül definierten Verifikationsregeln angezeigt.
    Mehr Informationen hierzu: Die Regelanzeige

  6. Eingabemaske für Vor-/Nachbedingungen

    Hier können Sie die zu den Knoten im Beweisbaum gehörenden Vor- bzw Nachbedingungen eingeben und verifizieren lassen.
    Mehr Informationen hierzu: Die Bedingungs-Eingabe

Anzeigeoptionen in der Toolbar

Mittels des Buttons "Details im Baum ein/ausblenden" können Sie für den markierten Knoten den Quellcode direkt im Baum anzeigen lassen.

Aufteilung der Bereiche

Die Aufteilung zwischen den einzelnen Bereichen kann mit einem Schiebebalken verändert werden. Per Klick auf die Kante können Sie die Grenze verschieben.

Beamer - Modus

Der Beamer-Modus ermöglicht das schnelle Einstellen von Anzeigeoptionen, die die Präsentation in Vorlesungen oder ähnlichen Veranstaltungen begünstigen. Dieser Modus ist zu finden unter "Hoare" => "Beamermodus".
Ist der Modus aktiv, so erscheint vor diesem Menüeintrag ein Häkchen. Um den Modus wieder auszuschalten, entfernen Sie einfach den Haken per Klick.

Der Beweisbaum

Der Beweisbaum wird vom Nutzer Stück für Stück aufgebaut um das gewünschte C0-Programm zu verifizieren. Der gewählte Knoten wir dabei hervorgehoben.

Der Quellcode

In dieser Anzeige wird der Quellcode dargestellt. Dieser kann jederzeit geändert und neu eingelesen werden.
Bei Auswahl eines Knotens im Beweisbaum, wird das dazugehörige Quellcode-Stück hervorgehoben.

Der Event-Log

Hier werden relevante Ereignise und Fehlermeldungen ausgegeben, unter Anderem:

  • Fehler beim Parsen des C0-Programms
  • Syntaktische Fehler bei der Eingabe der Vor/Nachbedingungen
  • Fehler bei falscher Regelanwendung
  • Evaluationsfehler bzw -Erfolg

Die Regelanzeige

Hier werden die für das Hoare-Kalkül definierten Verifikationsregeln angezeigt, welche beim Klicken darauf auch auf den vorher gewählten Knoten angewendet werden können.

Die Bedingungs-Eingabe

Hier werden die Vor/Nachbedingung eines Knotens eingegeben. An Knoten, bei denen diese sich automatisch ergeben wird dies vom Programm übernommen.
