Der Willkommensbildschirm

Hoare-Modul starten

Nach Starten des Hauptprogramms j-Algo können Sie über den Button "Neu" oder mit dem Menüpunkt "Datei" => "Neu" eine neue Instanz des Hoare-Moduls öffnen. Anschließend öffnet sich der Willkommensbildschirm des Moduls, der Ihnen verschiedene Möglichkeiten eröffnet.

Laden eines C0-Programmes aus einer Datei

Mit Klick auf das erste Symbol gelangen Sie in einen Dateiauswahl-Dialog, in welchem Sie ein C0-Programm laden können, welches dann verifiziert werden soll.
Siehe: Modulfunktionen

Eingeben eines C0-Programmes

Mit Klick auf das zweite Symbol gelangen Sie direkt zum Arbeits-Bildschirm des Moduls, in welchem Sie ein C0-Programm eigenständig in das dafür vorgesehene Textfeld eingeben können.
Siehe: Modulfunktionen

Laden eines Beweisbaumes aus einer Datei

Mit Klick auf dieses Symbol gelangen Sie in einen Dateiauswahl-Dialog, in welchem Sie einen gespeicherten Beweisbaum aus einer .jalgo-Datei laden können. Siehe: Modulfunktionen
